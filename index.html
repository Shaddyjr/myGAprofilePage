Work Experience

All Star Code

I'm working full-time as the Curriculum Manager at All Star Code (ASC), where I've already successfully lead the 2016 Summer Intensive. ASC is a non-profit strives to foster entrepreneurship and coding skill in black male youths in order to offset the underrepresentation of men of color in the tech industry. Though challenging, the work is incredibly rewarding. Teaching the students at ASC has greatly improved my own technical skill and I've been humbled by my time at ASC, where celebrating failure is a catalyst for learning.

ScholarStem

I'm also working part-time as the Curriculum Developer at ScholarStem (SS), where I've been working for a year. SS is a for-profit organization looking to fill the education need for coding in schools for grades 3-8 (both with instructors and as professional development for current teachers). ScholarStem uses a kid-freindly curriculum with interactive lessons about Scratch to introduction coding, while slowly ramping up to Javascript. For the older grades, Javascript skills are more fully realized and the curriculum culminates in a full on Javascript library capstone project.

Having taught children before, working with SS was my first experience teaching coding to middle-schoolers. Its was an enlightening and enjoyable experience. I wish I could of had these classes when I was their age!
